# Setup Instructions

## Hardware Setup
1. Connect energy harvesting tile output to ESP32 analog pin.
2. Connect ground to ESP32 GND.
3. Power ESP32 via USB.

## Software Setup
1. Install Arduino IDE.
2. Install ESP32 board package.
3. Upload the ESP32 code from src/esp32_code.ino.

## Running the Dashboard
1. Connect ESP32 to WiFi.
2. Open the ESP32 IP address in browser.
3. The web dashboard will display live voltage data.
4. will turn off the circuit after certain voltage threshold.
