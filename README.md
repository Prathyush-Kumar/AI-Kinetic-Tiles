# Energy Harvesting Smart Floor Tile System

## Overview
This project demonstrates an energy harvesting system that captures mechanical energy from human footsteps using smart tiles and monitors the generated voltage using an ESP32 microcontroller.

## Features
- Footstep energy harvesting
- ESP32 based voltage monitoring
- WiFi data transmission
- Real-time web dashboard

## Hardware Used
- ESP32
- Energy harvesting tiles
- Voltage conditioning circuit

## Software Used
- Arduino IDE
- HTML
  

## Output
The system generates voltage when pressure is applied on the tiles and displays real-time data on a web interface and prevent loss of energy by controlling power flow.
